3 rules of using JSX for creating element in React

  1. Only return 1 element (if you want to create multiple element you should use pair enclosing )

  2. Always close all element tag, if you forget to close element tag it will give an error "Unterminated JSX contents"

  3. Use camelCase for attribute element for example :
    Instead using class attribute in HTML, in JSX we are using className
    Instead using onclick attribute in HTML, in JSX we are using onClick
